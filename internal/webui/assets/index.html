Raw JSON config
Full editor · every quiccochet field is reachable from here

Edit any field directly — this is the exact JSON written to /etc/ispof/tunnels/<name>.json. quiccochet ignores unknown keys, so panel-managed metadata (_ispof_imported_from, created_at) round-trips safely. Changing the name field is not allowed; rename by creating a new tunnel and deleting the old one. After saving, restart the tunnel for the new config to take effect.

To change these

Edit /etc/default/ispof and run systemctl restart ispof. Available env vars:

ISPOF_LISTEN=0.0.0.0:3000 # bind address:port ISPOF_LOG_LEVEL=info # debug | info | warn | error ISPOF_AUTH=admin:<password> # basic auth (empty = no auth, loopback only) ISPOF_SCRAPE_INTERVAL=10s # Prometheus scrape cadence ISPOF_TUNNELS_DIR=/etc/ispof/tunnels # where configs are stored

About Ispof
High-performance L3/4 tunneling proxy with bidirectional IP spoofing

Ispof is a tunneling proxy with mutual IP spoofing and QUIC transport, designed to bypass DPI and stateful firewalls in restrictive network environments.
